# obs_led_clock
a led clock  plugin for obs

OBS　LED CLOCK

一个OBS实时显示当前时间的LED时钟插件

安装：

1.复制led-clock文件夹和文件 led-clock.lua 到 OBS 脚本目录：

C:\Program Files (x86)\obs-studio\data\obs-plugins\frontend-tools\scripts

2.打开OBS，菜单 “工具”-》“脚本”，点击右下的“+”按钮，加载 led-clock.lua

3.在OBS场景 添加来源 LED Clock,确定即可。

当前测试OBS版本为 23.0.2 使用通过
